About This Application

This application uses machine learning (YOLO) to identify empty rooftops in Portugal that are suitable for photovoltaic (PV) panel installations.

How to use:
  1. Use the search box to find a specific location or address in Portugal
  2. Select an area on the map using the drawing tool
  3. Ensure the area is under the maximum allowed size
  4. Click "Analyze Area" to detect empty rooftops
  5. View the results on the map and in the statistics panel
Technology:
  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript
  • Mapping: Google Maps API
  • Rooftop Detection: YOLO (You Only Look Once)
  • Solar Calculation: PVLib
  • GIS Processing: GeoPandas & Shapely
  • Visualization: Chart.js
